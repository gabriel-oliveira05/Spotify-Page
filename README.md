# Spotify_Page

## Descrição

Este projeto é uma aplicação desktop, desenvolvida com **HTML5, CSS3, JavaScript**, para fins de treinamento. Essa página mostra uma cópia do aplicativo **Spotify** que desenvolvi com a ajuda do canal DevClub no Youtube. A estrutura desse site desenvolvi junto com o video, mas adicionei alguns elementos amais na página, como por exemplo: Os botões acima **Inscrever-se** e **Entrar** adicionei depois, e adicionei também a **cesta** de compras na barra de pesquisa e também adicionei um **Transform: scale(1.1)** no **hover** das imagens. Imagens adicionadas no site com JavaScript usando **array** criando uma "div" e jogando as imagens com os nomes direto para o HTML.
