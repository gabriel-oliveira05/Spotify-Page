# Spotify_Page

## Descrição

Este projeto é uma aplicação desktop, desenvolvida com *HTML5, CSS3, JavaScript*, para fins de treinamento. Essa página mostra uma cópia do aplicativo *Spotify* que foi feito com a ajuda do canal DevClub no Youtube. A estrutura desse site desenvolvi junto com o video, mas adicionei alguns elementos a mais na página, como por exemplo: Os botões acima *Inscrever-se* e *Entrar* adicionei depois, e adicionei também a *cesta* de compras na barra de pesquisa, e também adicionei um *Transform: scale(1.1)* no *hover* das imagens pra deixar as imagens animadas quando passar o cursor em cima. Imagens adicionadas no site com JavaScript usando *array*, depois foi criado uma "div" dentro do JS e jogando as imagens com os nomes dos cantores direto para o HTML.
